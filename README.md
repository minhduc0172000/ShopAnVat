# ShopAnVat

ShopAnVat is a web application for selling packaged snacks online.

## Description
This project is a simple e-commerce website where users can browse snack products, add items to a cart, and place orders.

## Features

- User login and registration
- Product listing
- Shopping cart
- Order management
- Admin panel for managing products

## Tech Stack

- PHP
- MySQL
- HTML
- CSS
- JavaScript
